# Weather App

Search for a location and display the weather information in celsius or fahrenheit.

[Check Weather!](https://nirmalsubedi.github.io/odin-weather-app/)

## Which skills were demonstrated?

- Using the visual crossing timeline weather API to retrieve weather data.

- Working with the fetch API to retrieve data from visual crossing.

- Using linter and formatter to keep consistent codebase.

- Interacting with Promise based asynchronous code.

- Manage asynchronous task with async and await keyword.

- Handling any errors by creating a higher order function for asynchronous function which centralizes error handling.
